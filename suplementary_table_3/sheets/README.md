# Supplementary Table 3 — reproducible CPU/RAM pipeline

This directory rebuilds the RFhap versus hifiasm-trio CPU/RAM workbook directly
from the original Nextflow execution reports and yak/hifiasm stderr logs.

## Inputs

Paths are controlled by `config.json` and are resolved relative to this folder:

- `../rfhap/pipelines_rfhap_info`: RFhap Nextflow traces and HTML reports.
- `../rfhap/assemblies`: hifiasm-ONT logs for the RFhap-partitioned haplotypes.
- `../hifiasm`: yak parental-database and hifiasm-trio logs.

No intermediate TSV generated by the older analysis is required.

## Reproduce

```bash
cd suplementary_table_3/sheets
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
python run_pipeline.py --clean
```

The only Python dependency is `XlsxWriter`.

## Outputs

All generated files are written to `outputs/`:

- `Supplementary_Table_3_CPU_RAM_Numbers.xlsx`: final workbook, one sheet per dataset.
- `Supplementary_Table_3_CPU_RAM_components.tsv`: component, stage-total, and end-to-end rows.
- `Supplementary_Table_3_CPU_RAM_totals.tsv`: stage and workflow totals.
- `trace_selection_qc.tsv`: all candidate RFhap traces and the selected run.
- `selected_traces.tsv`: one selected complete run per dataset.
- `parsed_external_logs.tsv`: parsed yak/hifiasm command metrics.
- `qc_summary.json`: dataset coverage, duplicate/missing checks, and formula counts.

## Calculation rules

- RFhap component CPUs: maximum task allocation reported by Nextflow.
- RFhap stage CPUs: peak concurrent Nextflow CPU allocation, compared with the
  requested threads of standalone haplotype assemblies.
- yak/hifiasm CPUs: maximum requested `-t` threads within each stage.
- Peak RSS: maximum reported resident memory; memory is never summed.
- Workflow totals: maximum stage CPU/thread and RSS requirements.
- The workbook contains live `MAX` formulas and cached results for immediate
  display in Excel or Apple Numbers.

The pipeline stops with an error if a dataset lacks an expected process, contains
duplicate process roles, lacks a complete trace, or produces a sheet with fewer
than 12 calculated cells.
